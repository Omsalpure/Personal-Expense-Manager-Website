# 💰 Personal Expense Manager

## 📌 Overview

The Personal Expense Manager is a web-based application that helps users track and manage their daily expenses efficiently. It provides a secure and user-friendly interface to add, edit, delete, and view expenses in one dashboard.

---

## 🚀 Features

* 🔐 User Registration & Login (Secure Authentication)
* ➕ Add New Expenses
* ✏️ Edit Existing Expenses
* ❌ Delete Expenses
* 📊 Dashboard with Total Expense Calculation
* 📂 Category-wise Expense Tracking
* 👤 User-specific Data (Privacy Protected)

## ⚙️ Installation & Setup

1. Install XAMPP and start Apache & MySQL

2. Copy project folder to:

   ```
   C:\xampp\htdocs\
   ```

3. Open phpMyAdmin and run the SQL script to create database:

   * Database name: `expense_tracker`

4. Import tables (users & expenses)

5. Open browser and run:

   ```
   http://localhost/expense_pro/register.php
   ```
